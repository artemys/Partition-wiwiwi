# TabScore Web

Frontend Next.js pour piloter les transcriptions audio du backend FastAPI.

## Démarrage local

```bash
pnpm install
pnpm dev
```

Ouvrir `http://localhost:3000`.

## Variables d’environnement

Créer un `.env.local` si besoin :

```
NEXT_PUBLIC_API_URL=http://localhost:8000
```

## Build

```bash
pnpm build
pnpm start
```
